# AI Avventura

Applicazione educativa gamificata per insegnare i fondamenti dell'AI generativa a bambini di 9 anni.

## Quick Start

cp .env.example .env
docker compose up --build

Apri http://localhost:8080

## Accessi

- Bambini: codice sessione + nome squadra
- Admin: http://localhost:8080/admin
- Demo: http://localhost:8080/demo

## Personalizzare le domande

Modifica i file JSON in content/ e riavvia il container.
